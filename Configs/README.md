# ⚙️ Configuration Files

Collection of common configuration files for development projects.

## Files Included

- `.editorconfig` - Editor configuration for consistent coding styles
- `.prettierrc.json` - Prettier code formatter configuration
- `.eslintrc.json` - ESLint linting configuration
- `.gitignore` - Git ignore patterns

## Usage

Copy these files to your project root and customize as needed.
